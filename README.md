# LPG Leak Monitoring System
## Overview
This project was developed as an undergraduate thesis project to monitor LPG gas leakage using IoT technology and Fuzzy Mamdani decision-making.

## Components
- MQ-5 Gas Sensor
- DHT11 Temperature Sensor
- Arduino
- MQTT
- Node-RED

## Features
- LPG gas leakage detection
- Temperature monitoring
- Buzzer alarm activation
- Exhaust fan control
- Email notification system
